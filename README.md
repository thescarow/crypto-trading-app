## Crypto Trading App – CBB Assessment

Simple crypto trading demo built with **React + TypeScript**, **Vite**, **React Query**, **Zustand**, and plain CSS.

### Run locally

1. `npm install`
2. `npm run dev`
3. Open `http://localhost:5173`

### Features

- Sticky header with navigation and login (local-only)
- Home page: crypto assets table with sorting and load more
- Trade page (login required): crypto ↔ USD swap-style form with live conversion
